About this app

This is an example application for Web3.Storage, a free service for decentralized data storage.

Web3.Storage is a service to make building on top of Filecoin as simple as possible - giving the developers the power of open, distributed networks via a friendly JS client library.

Behind the scenes, Web3.Storage is backed by Filecoin and makes content available via IPFS leveraging the unique properties of each network.

See the documentation to learn more!
